# ImageGen mouth sheet prompt

Create one production-ready character mouth-animation sprite sheet with a strict 3-column by 2-row layout and no gutters or grid lines.

Use the attached character portrait only as the design reference for the human character. Generate both characters as new artwork in one unified polished chibi anime style: bold clean outlines, soft cel shading, warm approachable expressions, bust portraits, identical scale, identical lighting, and a plain warm off-white background.

Top row: the exact same original red Shiba Inu character in every cell, facing three-quarter right toward the center, wearing a small teal scarf. Column 1 mouth fully closed, column 2 mouth slightly half-open, column 3 mouth clearly open as if speaking.

Bottom row: the exact same Itopan human character in every cell, based on the attached portrait: warm tan skin, black hair tied in a top bun, closely shaved side, brown eyes, plain white T-shirt, facing three-quarter left toward the center. Column 1 mouth fully closed, column 2 mouth slightly half-open, column 3 mouth clearly open as if speaking.

Within each row, preserve the exact same face shape, eyes, hair, ears, clothing, body pose, crop, camera angle, lighting, and proportions. Only the mouth changes. Do not add words, labels, letters, logos, speech bubbles, borders, dividers, props, scenery, or extra characters. Each of the six cells must be an equally sized square portrait that can be cropped mechanically.
