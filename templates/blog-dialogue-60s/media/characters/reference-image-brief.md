# Character reference brief

## 共通シート

- ImageGenで3列×2行の1枚を生成する。
- 上段はオリジナル赤柴、下段は添付画像を参照して新規生成するイトパン。
- 3列は口閉じ、半開き、開き。各行で口以外の顔、髪、服、姿勢、画角、照明を変えない。
- 太めの線、柔らかいセル塗り、チビキャラの胸像、淡い無地背景で統一する。
- 文字、罫線、ロゴ、吹き出し、背景小物を入れない。

## Shiba

- オリジナルの赤柴。左に置くため右向き3/4で中央を見る。
- 識別要素は青緑の小さなスカーフ。

## Itopan

- 添付画像の団子ヘア、刈り上げ、褐色の肌、白Tシャツを識別要素にする。
- 右に置くため左向き3/4で中央を見る。
- 添付画像そのものはgit管理外とし、生成した3口形だけを動画素材にする。
